0x01  
学习一个漏洞的原理时，一定要考虑这个漏洞通常出现在什么样的开发架构中，这样的开发架构通常出现在什么样的业务中，每次积累一点，久而久之，最终成长为看到一个业务大概就知道存在什么漏洞

0x02  
带团队时，对实习生的建议：  
掌握漏洞的基本原理  
掌握漏洞的利用方式  
掌握测试工具的使用
