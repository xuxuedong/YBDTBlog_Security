前一天强制关机，今天早上开机提示“ext4 read failed ...”

解决方案：  
再次强制关机，开机后会进入启动项选择界面，选择“Advanced options for ubuntu”，进入子菜单后会有4个选项  
linux ...  
linux ... recovery mode  
linux ...  
linux ... recovery mode  
选择第3个，第3个是最后一次正确的内核，即可成功进入系统
